This plugin allows the connection between the Chilean payment gateway through As a Service Labs.

== Description ==

This service makes reciving payments in Chile really easy. The platform is already certified so you don't need to certified your store to start receiving money.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Get the Tokens from : https://panel.cristiantala.cl/
1. Use the Woocommerce -> Settings-> Checkout->Transbank As A Service to configure the plugin
1. Use your token_service and token_secret in the config.
